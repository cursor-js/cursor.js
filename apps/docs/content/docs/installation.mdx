---
title: Installation
---

You can install `cursor.js` using your preferred package manager:

<Tabs items={["npm", "yarn", "pnpm"]}>
  <Tab value="npm">
```bash
npm install @cursor.js/core
```
  </Tab>
  <Tab value="yarn">
```bash
yarn add @cursor.js/core
```
  </Tab>
  <Tab value="pnpm">
```bash
pnpm add @cursor.js/core
```
  </Tab>
</Tabs>
